## 🌧️ Rain Detection System using ESP8266

## 📌 Project Overview
The Rain Detection System is an IoT project developed using the ESP8266 (NodeMCU), a Rain Sensor, LED, and Buzzer. The system detects rain in real time and displays the rain status on a web page. When rain is detected, the LED turns ON and the buzzer sounds as an alert.

## 🎯 Objective
Detect rainfall using a rain sensor.
Indicate rain status using an LED and buzzer.
Display rain status on a web server.
Learn IoT monitoring using ESP8266.

## 🛠️ Components Required
ESP8266 NodeMCU,
Rain Sensor Module,
LED,
220Ω Resistor,
Active Buzzer,
Breadboard,
Jumper Wires,
USB Cable,
Wi-Fi Connection


## ⚙️ Working Principle
The rain sensor continuously checks for water droplets.
ESP8266 reads the sensor's digital output.

## If rain is detected:
🌧️ LED turns ON.
🔊 Buzzer sounds.
🌐 Web page displays "Rain Detected".
If no rain is detected:
💡 LED turns OFF.
🔕 Buzzer turns OFF.
🌐 Web page displays "No Rain".

## ▶️ How to Run
Connect all components as shown in the circuit connections.
Open the project in Arduino IDE.
Install the ESP8266 Board Package.
Enter your Wi-Fi SSID and Password.
Upload the code to the ESP8266.
Open the Serial Monitor.
Copy the IP Address displayed.
Open the IP Address in a web browser.
Test the rain sensor by adding a few drops of water
.
